### 1.2 实验

（1）在VC++中解决方案是最大的文件（夹）管理系统，不同的C++项目集成在解决方案下，每个项目都有源文件、相关资源文件。此外，解决方案是随着第一个项目的创建自动生成的。

（2）使用Visual C++创建项目、书写源程序。

（3）通过使用断点调试并测试程序。可以：1、在界面双击；2、快捷键。

（4）关于程序的发布：首先，清理早期生成过程中创建的中间文件和输出文件，确保是最终版本；然后，在状态栏将Debug改为Release，生成解决方案即可。
